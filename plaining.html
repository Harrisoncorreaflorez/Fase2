Contenido en HTML

SECTION:

El section elemento representa una sección genérica de un documento o aplicación. Una sección, en este contexto, es una agrupación temática de contenido, normalmente con un encabezado. Ejemplos de secciones serían los capítulos, las distintas páginas con pestañas en un cuadro de diálogo con pestañas o las secciones numeradas de una tesis. La página de inicio de un sitio web se puede dividir en secciones para una introducción, noticias e información de contacto. Se anima a los autores a utilizar el article elemento en lugar del section elemento cuando tenga sentido distribuir el contenido del elemento. El section elemento no es un elemento contenedor genérico. Cuando se necesita un elemento para fines de estilo o como conveniencia para la creación de secuencias de comandos, se anima a los autores a utilizar el div elemento en su lugar. Una regla general es que el section elemento es apropiado solo si el contenido del elemento se enumeraría explícitamente en el esquema del documento.

ASIDE:

El aside elemento representa una sección de una página que consta de contenido que está tangencialmente relacionado con el contenido alrededor del aside elemento y que podría considerarse separado de ese contenido. Estas secciones a menudo se representan como barras laterales en tipografía impresa. El elemento se puede utilizar para efectos tipográficos como comillas extraídas o barras laterales, para publicidad, para grupos de nav elementos y para otro contenido que se considera separado del contenido principal de la página. No es apropiado usar el aside elemento solo para paréntesis, ya que son parte del flujo principal del documento.

FOOTER:

El Elemento HTML Footer representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección (p.e, su ancestro mas cercano article, aside, nav, section,blockquote, body, details, fieldset, figure, td). Un pie de página típicamente contiene información acerca del autor de la sección, datos de derechos de autor o enlaces a documentos relacionados.

ARTICLE:

El Elemento article de HTML representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable, por ejemplo, en la indicación. Podría ser un mensaje en un foro, un artículo de una revista o un periódico, una entrada de blog, un comentario de un usuario, un widget interactivo o gadget, o cualquier otro elemento independiente del contenido.
